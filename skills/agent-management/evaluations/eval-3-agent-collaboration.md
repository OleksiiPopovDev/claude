# Evaluation 3: Agent Collaboration Design

## Scenario

User is designing an agent that needs to collaborate with other agents.

## Test Prompt

"I'm creating a Frontend Developer agent. How should I design the collaboration and handoff points with other agents like the Backend Developer and QA Engineer?"

## Expected Behavior

Claude should:
1. Activate the `agent-management` Skill
2. Explain handoff instruction structure
3. Suggest what artifacts to share with Backend (API specs, endpoints)
4. Suggest what to provide to QA (test requirements, coverage expectations)
5. Show XML or Markdown structure for collaboration sections
6. Emphasize clear boundaries between agent roles

## Success Criteria

- [ ] Skill activates correctly
- [ ] Explains handoff instruction concept
- [ ] Suggests specific artifacts for Backend handoff
- [ ] Suggests specific artifacts for QA handoff
- [ ] Shows structure example (XML or Markdown)
- [ ] Discusses clear role boundaries
- [ ] Provides actionable collaboration design
